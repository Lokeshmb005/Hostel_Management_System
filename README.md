# 🏛️ Smart Hostel Management System

A fully deployed web application that digitizes and unifies hostel operations for both **Students** and **Wardens** — built with HTML, CSS & JavaScript.

🌐 **Live Demo:** [https://sec-hostel.netlify.app](https://sec-hostel.netlify.app)

## ✨ Features

- 🎓 **Dual Role System** — Separate portals for Students and Wardens
- 📋 **Real-time Attendance Tracking** — Monitor student attendance digitally
- 🚪 **Digital Gate Pass Approval** — Request and approve gate passes online
- 🔔 **Instant Notifications & Alerts** — Stay updated with real-time alerts
- 📊 **Live Floor Status Dashboard** — Visual overview of hostel floor activity
- 🔐 **Secure Authentication** — Google Auth + Email/Password login powered by **Firebase Auth**
- 🗄️ **Cloud Database** — User data securely stored and retrieved using **Firebase Firestore**
- 📝 **Full Registration & Onboarding Flow** — Easy account setup for students and wardens

## 👤 User Roles

### 🎓 Student
- Register with room number, floor, and parent contact
- Request gate passes
- View attendance records
- Receive hostel notifications

### 🛡️ Warden
- Approve or reject gate pass requests
- Monitor real-time floor status
- Track student attendance
- Send notifications and alerts
